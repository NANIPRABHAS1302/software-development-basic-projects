# Software Development Basic Projects
A collection of beginner-friendly software development projects developed using HTML, CSS, and JavaScript as part of the Cognifyz Software Development Internship.
## Projects Included
### 1. Cyber Quiz Challenge
An interactive quiz-based game with score tracking, responsive UI, and modern design.
### 2. Pattern Generator Pro
A dynamic pattern generator capable of creating multiple number and star patterns with an advanced interface.
### 3. TaskFlow Pro CRUD Application
A smart task management system implementing Create, Read, Update, and Delete (CRUD) operations with local storage support.
### 4. TempSync Pro
An advanced temperature converter supporting Celsius, Fahrenheit, and Kelvin conversions with responsive UI design.
